# AI 项目仓库

这个仓库包含了AI相关的项目和学习资源。

## 📁 项目内容

### 🎮 俄罗斯方块网页游戏

一个使用HTML、CSS和JavaScript开发的经典俄罗斯方块游戏。

#### 特性
- 🎯 经典俄罗斯方块玩法
- ⌨️ 键盘控制（方向键移动，上键旋转）
- 📱 响应式设计
- 🎨 彩色方块和简洁界面
- 🏆 分数统计

#### 如何运行
直接在浏览器中打开 `tetris.html` 文件即可开始游戏。

#### 操作说明
| 按键 | 功能 |
|------|------|
| ← (左箭头) | 向左移动方块 |
| → (右箭头) | 向右移动方块 |
| ↓ (下箭头) | 加速下落 |
| ↑ (上箭头) | 旋转方块 |

## 🚀 快速开始

1. 克隆仓库
   ```bash
   git clone https://github.com/jianghx1997-sudo/AI.git
   ```

2. 进入项目目录
   ```bash
   cd AI
   ```

3. 打开俄罗斯方块游戏
   ```bash
   # Windows
   start tetris.html
   
   # macOS
   open tetris.html
   
   # Linux
   xdg-open tetris.html
   ```
